FAQ

How do I join a ride? How do I leave a ride? How do I create a new ride? What is a secret code? How do I edit a ride I made? How do I edit a ride I didn't make? How do I help make this site better?
Click the "Join" button in the row of the ride you want to join. After filling in your name and phone number on the following page, you should see your name in a passenger spot on the main list. If the car is full, you cannot join it. Coming soon! Fill in the information on the right side of the page and click "Add Ride". A secret code is a code needed to edit your ride. As this is still a proof of concept, we don't have encryption set up yet, so do NOT use a password you've used on any other site. Type your secret code in the box next to the edit button then click the edit button. If you did not make a secret code, just click the edit button. Ask the person who made it for the secret code. We are so glad you want to help! Right now, the site is in a proof of concept stage so the code is a bit of a mess. Given this goes well, it will be updated to a more scalable design. After the update, we welcome your contributions. Here is the source code. Please submit your ideas as an issue and we'll update this section when we are open to pull requests.
